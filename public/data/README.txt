Siyo! This is a package for the Cherokee Reference Dictionary app. Please be careful editing these files directly. They may not import into the app correctly if modified.

The contents of a package are:
/metadata.json - Package metadata
/README.txt - README
/dictionary.csv - Dictionary entries
/sentences.csv - Sentence entries
/join_table.csv - Sentence-to-entry gloss data
/audio/__.mp3 - Audio files   

The 3 .csv files contain the word and sentence data in a package. They are formatted as follows:
dictionary.csv:
Index - The index of the entry
Source - The shorthand source name
Entry - The transliteration of the word
Syllabary - The syllabary of the word
Definition - The English definition of the word
Cross_Reference - List of "related words"
